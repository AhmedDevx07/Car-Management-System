# 🚗 Car Management System

A responsive and user-friendly **Car Management System** built with **HTML**, **Bootstrap 5**, and **vanilla JavaScript**.
This project allows users to **add**, **view**, **search**, and **delete** car listings in an interactive way.

---

## 🌟 Features

* ➕ Add new cars with details (name, model, price, color, address, status)
* 📋 View all cars dynamically
* 🔍 Search cars by name
* ❌ Delete cars instantly
* 🧹 Clear all displayed data
* 💎 Fully responsive UI using **Bootstrap 5**

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3 (Bootstrap 5)**
* **JavaScript (ES6)**

---

## 🧩 How It Works

1. Enter car details in the form
2. Click **“Add Car”** to display it below
3. Click **“Show All Cars”** to view all cars from the array
4. Use **Search** to find cars by name
5. Click **“Delete”** to remove a car card instantly
